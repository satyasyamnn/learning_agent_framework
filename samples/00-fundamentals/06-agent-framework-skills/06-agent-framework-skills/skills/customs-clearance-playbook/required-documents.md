# Required Documents

Use this checklist before filing customs entry:

- Commercial invoice with buyer, seller, Incoterms, line-item values, and currency.
- Packing list with package count, carton IDs, and gross/net weight.
- Bill of lading or airway bill.
- Country of origin declaration or certificate when preference is claimed.
- Import permit/license if commodity is controlled.
- Entry summary draft with HS classification and valuation method.

Escalate if any mandatory field is missing, contradictory, or unverifiable.
