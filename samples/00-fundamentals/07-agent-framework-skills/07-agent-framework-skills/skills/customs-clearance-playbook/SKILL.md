# Customs Clearance Playbook

Use this skill when the user asks about customs entry readiness, missing documentation, risk routing, or hold/release recommendations.

## How to Use

1. Load this skill for customs process questions.
2. Read resource files to ground your answer in the playbook.
3. Explain what is complete, what is missing, and the next operational step.

## Required behavior

- Do not invent regulations that are not in the referenced resources.
- Call out mandatory controls before release.
- Prefer concise, operational recommendations.
