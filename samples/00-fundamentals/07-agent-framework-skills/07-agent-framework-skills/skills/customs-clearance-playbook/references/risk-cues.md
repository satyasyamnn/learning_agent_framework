# Risk Cues

Common customs review cues:

- Origin/transshipment route is sanctions-sensitive.
- HS code suggests controlled or dual-use products.
- Declared value appears inconsistent with commodity profile.
- Missing certificate of origin where tariff preference is claimed.
- Import license required but not present.
- Restricted-party screening result is unresolved.

Lane guidance:

- Green: No critical issues, documentation complete.
- Amber: Gaps requiring analyst review before release.
- Red: Critical compliance controls missing or restricted-party concerns.
