# 01-RAG Samples

This folder contains Retrieval-Augmented Generation (RAG) examples built with Microsoft Agent Framework.

## Current Samples

- `00-customs-rag-basic`: Basic customs-domain RAG using an in-memory knowledge base and retrieval tool.
- `01-customs-rag-embeddings`: Embedding-based customs RAG using semantic vector retrieval with in-memory vector store.

## Planned Advanced Samples

Additional projects will be added here for advanced RAG patterns such as:
- Chunking and indexing strategies
- Hybrid retrieval (keyword + semantic)
- Re-ranking
- Multi-step retrieval workflows
- Grounding and citation validation
